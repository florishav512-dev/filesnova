Frequently Asked Questions

Are my files safe when using Files Nova?

Yes, absolutely! All file processing happens directly in your browser. Your files are never uploaded to our servers, ensuring complete privacy and security. Once you close your browser tab, all traces of your files are removed from memory.

What file formats do you support?

We support a wide range of formats including:

  • Images: JPG, PNG, GIF, BMP, WEBP
  • Documents: PDF, DOCX, TXT
  • Archives: ZIP (creation only)

Is there a file size limit?

File size limits depend on your device's available memory. Generally, we recommend files under 100MB for optimal performance. For larger files, the processing may take longer or may not work on devices with limited RAM.

Do I need to create an account?

No account creation is required! All our tools are available for free without registration. Simply visit our website and start converting your files immediately.

Can I convert multiple files at once?

Yes, many of our tools support batch processing. You can select multiple files for conversion in tools like JPG to PDF, PNG to PDF, Image Compression, and Files to ZIP.

Why is my conversion taking a long time?

Processing time depends on several factors:

  • File size and complexity
  • Number of files being processed
  • Your device's processing power
  • Available system memory

For faster processing, try converting fewer files at once or reducing file sizes before conversion.

Tool-Specific Guides

PDF Tools

Merging PDFs: Select multiple PDF files and click "Merge PDFs" to combine them into a single document. Files will be merged in the order you select them.

Splitting PDFs: Upload a PDF and specify page ranges (e.g., "1-3, 5, 7-9") to extract specific pages.

PDF Unlock: Remove password protection by uploading the PDF and entering the correct password.

Image Tools

Image Compression: Reduce file sizes while maintaining quality. Use the quality slider to balance file size and image quality.

Image Resizing: Change image dimensions by entering width and height values. Check "Maintain aspect ratio" to prevent distortion.

Image to PDF: Convert JPG or PNG images to PDF format. Multiple images will create a multi-page PDF.

Text Tools

Text to PDF: Enter text in the textarea and generate a formatted PDF document.

Word Counter: Get real-time statistics including word count, character count, and paragraph count.

Troubleshooting

Tool not responding or stuck?

  • Refresh the page and try again
  • Check if your file size is too large
  • Ensure your browser supports JavaScript
  • Try using a different browser (Chrome, Firefox, Safari, Edge)

Error messages appearing?

  • Check that you've selected the correct file format
  • Ensure your file isn't corrupted
  • Try with a smaller file first
  • Clear your browser cache and cookies

Download not starting?

  • Check if your browser is blocking downloads
  • Disable popup blockers for our site
  • Try right-clicking the download link and selecting "Save as"

Browser Compatibility

Files Nova works best with modern browsers that support the latest JavaScript features:

  • Google Chrome 80+
  • Mozilla Firefox 75+
  • Safari 13+
  • Microsoft Edge 80+

For optimal performance, we recommend keeping your browser updated to the latest version.
